##Rock Paper Scissors Game with Tkinter GUI

This is a classic game of Rock Paper Scissors built using the Tkinter GUI toolkit in Python. The game allows to play against the computer.

The front-end of the application is built using Tkinter and provides a user-friendly interface for players to select their choice of rock, paper, or scissors. The interface also displays the current score and the winner of each round. 

The back-end of the application handles the game logic and determines the winner of each round based on the player's choices. The application keeps track of the score and displays the winner of the game after a certain number of rounds. 

This application is useful for anyone who wants to play a quick game of Rock Paper Scissors with a friend or family member. It demonstrates how to build a basic game with Tkinter and can be used as a starting point for developing more complex games.

Installation:
	1. Clone this repository
	2. Run the Code in the downloaded folder.
	3. Play the game by selecting rock, paper, or scissors and waiting for the result.
